##  Quick Reference

### Key Formulas

**Sequences:**
- Geometric: $a, ar, ar^2, \ldots, ar^n$
- Arithmetic: $a, a+d, a+2d, \ldots, a+nd$

**Summation:**
$$\sum_{j=m}^{n} a_j = a_m + a_{m+1} + \cdots + a_n$$

**Combinations:**
$$\binom{n}{r} = \frac{n!}{r!(n-r)!}$$

**Binomial Theorem:**
$$(x+y)^n = \sum_{j=0}^{n} \binom{n}{j}x^{n-j}y^j$$

**Probability:**
$$P(A|B) = \frac{P(A \cap B)}{P(B)}$$

**Handshake Lemma:**
$$2m = \sum_{v \in V} \deg(v)$$
